Requerimientos de Usuario y Administrador

Historial de Usuario

En este apartado se visualizar todos los registros que ha echo el usuario de manera practica.

Requerimientos para Ciudadano

Registro de Usuarios

  • El usuario debe ser capaz de cambiar su información básica (nombre, correo electrónico, contraseña).
  • Dar validación de correos electrónicos y contraseñas.

Inicio de Sesión

  • Los usuarios deben poder iniciar sesión con sus credenciales.
  • Opción para "recordar sesión".

Perfil de Usuario

  • Los usuarios pueden ver su historial de actividades.
  • Posibilidad de actualizar información personal.

Registro de Actividades

  • Registrar actividades relevantes (inicios de sesión, agendar cita, comentarios, etc.).
  • Mostrar la fecha y hora de cada actividad.

Filtrado y Búsqueda

  • Opción para filtrar el historial por tipo de actividad y rango de fechas.
  • Funcionalidad de búsqueda para encontrar actividades específicas.

Exportación de Datos

  • Los usuarios pueden exportar su historial en formatos comunes (CSV, PDF, etc.).

Requerimientos para Administradores

Acceso Administrativo

  • Interfaz de inicio de sesión exclusiva para administradores.
  • Autenticación de dos factores (opcional) para mayor seguridad.

    • Gestión de Usuarios

    • Visualizar y gestionar el historial de todos los usuarios.
    • Opción para activar, desactivar o eliminar cuentas de usuario.

    Análisis de Datos

    • Herramientas para generar informes sobre la actividad de los usuarios.
    • Gráficas y estadísticas sobre el uso del sistema.

    Control de Actividades

    • Capacidad para revisar, moderar o eliminar actividades específicas de usuarios.
    • Notificaciones de comportamientos sospechosos o reportes de usuarios.

    Configuración del Sistema

    • Opciones para ajustar configuraciones del sistema (políticas de privacidad, límites de actividad, etc.).
