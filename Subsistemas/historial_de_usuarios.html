    • Requerimientos por Usuario

    El Ciudadano debe ser capas de:

    1. Dar registros

        1.1 para poder cambiar su información básica (correo electrónico y contraseña).
        1.2 para validaciones de correos electrónicos y contraseñas.

    2. Dar inicios de Sesión

        2.1 con sus credencial.
        2.2 para "recordar sesión".

    3. Poder ver su Perfil de Usuario

        3.1 para ver su historial de actividades (PDF, Citas, etc)

    4. Dar Registro de Actividades

        4.1 para poder ver actividades relevantes (inicios de sesión, comentarios, etc.).
        4.2 para Mostrar la fecha y hora de cada actividad.

    5. Dar Filtrados y Búsquedas

        5.1 para la busqueda del historial por tipo de actividad, rango de fechas y actividades específicas.

    6. Dar Exportación de Datos

        6.1 para que puedan exportar su historial en formatos comunes (CSV, PDF, etc.).

    El Administrador debe ser capas de:

    1. Dar Acceso

        1.1 Exclusivo para administradores.
        1.2 De Autenticación en dos pasos (opcional) para mayor seguridad.

    2. Dar Gestión a los Usuarios

        2.1 para Visualizar el historial de todos los usuarios en caso de ser solicitada.
        2.2 para poder activar, desactivar algun usuario.

    3. Dar Análisis de Datos

        3.1 para generar informes sobre la actividad de los usuarios.
        3.2 para proporcionar Gráficas y estadísticas sobre el uso del sistema.

    4. Dar Control de Actividades

        4.1 para notificar de comportamientos sospechosos por medio de algun correo.

    5. Dar Configuración del Sistema

        5.1 para ajustar configuraciones del sistema (políticas de privacidad, etc.).
